# mesh-leyu-guide

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放各种独立的 HTML 页面，内容涵盖信息展示、工具页面、文档索引等。每个页面均为独立的静态文件，不依赖特定后端服务，便于直接查看和归档。

## 目录说明

```
.
├── index.html          # 仓库主页（可选，用于导航）
├── pages/              # 存放主要 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源（样式、图片、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：所有独立 HTML 页面存放于此，文件名可自定。
- `assets/`：页面共用的资源文件，按类型分目录整理。

## 页面归档说明

- 每个 HTML 页面均为自包含的静态文件，可直接通过浏览器打开。
- 页面之间无强依赖关系，可单独使用或通过仓库主页索引。
- 建议在页面内注明创建日期和简要用途，便于后续维护。

## 维护说明

- 新增页面：在 `pages/` 下添加 `.html` 文件，并更新 `index.html`（如有）。
- 修改页面：直接编辑对应 HTML 文件，注意保持资源路径一致。
- 删除页面：移除对应文件，并清理相关引用。
- 资源更新：如需修改公共样式或脚本，请确保不影响已有页面。

## 使用方式

1. 克隆或下载本仓库。
2. 直接打开 `pages/` 下的 HTML 文件即可浏览。
3. 若通过 GitHub Pages 部署，访问仓库 Pages 地址即可查看所有页面。

## 许可

本仓库内容仅供归档与展示，不涉及商业用途。具体页面可能引用第三方资源，请遵守相应许可协议。
